豆瓣电影信息爬取

  • 1.手里有别人写好的电脑爬取完整版，但是看不懂，先只写爬豆瓣的python中的json代码

  • 2.通过观察豆瓣的网站，得到了item的东西是依靠原网站中的(referer)的，听不懂就跳过吧=_=，Ctrl+F查找电影名可在item中找到，所以直接获取start_url(第一步).

  • 3.由于item_url中是start = 0,count = 18进行加的，所以将start_url中的start = {}，放入到parse_url中

  • 4.在item的Preview中可发现,subject_collection_items控制的不同图片信息.,所以需要获得json格式的东西

  • 5.最后把获得的json格式转化为python类型然后保存到douban.json中即可
